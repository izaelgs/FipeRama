Sobre a FipeRama

FipeRama é um projeto de fins educativos que consiste em desenvolver uma aplicação web utilizando a biblioteca de funções JavaScript em conjunto com o framework web Bootstrap.

Este projeto tem como objetivo exibir os dados sobre marcas de automóveis existentes na Tabela FIPE usando a “API de Consulta da Tabela FIPE” disponível em http://fipeapi.appspot.com/.
